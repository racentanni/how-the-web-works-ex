Questions

1. Hyper Text Transfer Protocol is how clients get data from, or send data to a server.
2. URL stands for Uniform Resource Locator which is an address for an internet Resource
3. Domain Name System, which converts URLs into IP addresses
4. The query string lets one to pass key value pairs into the URL in the format of ?key1=value&key2=value
5. Get: gets data from the server (most pages, search forms) Post: Sends data to the server (changes data on the server)
6. An HTTP response is the response from a server to a client which follows HTTP protocol.
7. Headers provide additional information about a request or the response ex: Request headers: Host, User-Agent, Accept, Cookie. Response headers: Content type, Last-modified, set cookie, cache control.
8.
a. The browser resolves the name into an ip address using DNS
b. The browser then makes a request to that IP address including headers.
c. The server sends a response in HTML with a status code (hopefully 200).
d. The browser makes a DOM from that HTML and finds any other resources if needed.
e. The browser makes additional HTTP requests for those resources and receives response from the server for each.
